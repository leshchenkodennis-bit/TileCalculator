# Tiler's Assistant

PWA-калькулятор для расчёта стоимости плиточных работ и материалов.

## Возможности

- Расчёт площади (пол + стены)
- Автоопределение формата плитки и цены укладки
- Дополнительные работы: форматирование, изготовление изделий
- Автоматический расчёт материалов (клей, затирка, СВП, грунтовка и др.)
- Inline-редактирование всех позиций
- Экспорт: PDF-смета, CSV, копирование
- Профиль мастера в PDF
- PWA: работает офлайн
