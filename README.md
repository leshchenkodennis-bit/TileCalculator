# ПлиткаCalc — PWA Калькулятор плиточных работ

## Файлы
- `index.html` — основное приложение
- `manifest.json` — PWA манифест
- `sw.js` — Service Worker (офлайн-кеш)
- `icon-192.png` / `icon-512.png` — иконки

## Быстрый запуск (GitHub Pages)

### 1. Создать репозиторий
```
1. Зайти на github.com → New repository
2. Имя: tilecalc (или любое)
3. Public
4. Create repository
```

### 2. Загрузить файлы
```
1. В репозитории нажать "uploading an existing file"
2. Перетащить ВСЕ 5 файлов из этой папки
3. Commit changes
```

### 3. Включить GitHub Pages
```
1. Settings → Pages
2. Source: Deploy from a branch
3. Branch: main, / (root)
4. Save
```

### 4. Готово!
Через 1-2 минуты сайт будет по адресу:
`https://ТВОЙ_ЛОГИН.github.io/tilecalc/`

## Установка на телефон

### Android (Chrome)
1. Открыть ссылку в Chrome
2. Меню (⋮) → "Добавить на главный экран"
3. Появится иконка как обычное приложение

### iPhone (Safari)
1. Открыть ссылку в Safari
2. Кнопка "Поделиться" (□↑) → "На экран Домой"

## Обновление
1. Отредактировать `index.html` или заменить файл в GitHub
2. В `sw.js` поменять номер версии: `CACHE_NAME = 'tilecalc-v6'`
3. Commit → все устройства получат обновление автоматически

## Альтернатива: Netlify
1. netlify.com → Sites → перетащить папку
2. Готово — ссылка выдаётся моментально
3. Обновление: перетащить папку заново

## Без хостинга
Файл `tilecalc.html` (в корневой папке выше) работает автономно —
просто открыть в браузере. Без PWA-фич (иконка, офлайн-кеш),
но все функции калькулятора работают.
