# ПлиткаCalc — Калькулятор плиточных работ

PWA-калькулятор для расчёта стоимости плиточных работ и материалов.

## Возможности
- Расчёт площади (пол + стены)
- Работы: укладка, подрезка, затирка, герметизация и др.
- Материалы: клей, грунтовка, СВП, затирка, силикон и др.
- Автоподбор гребёнки по размеру плитки
- Экспорт: PDF-смета, CSV, текст
- История смет
- Работает офлайн (PWA)
